uTorrent Auto Extractor
====================

### About

uTorrent Auto Extractor is small opensource and free software
which is used to automatically copy and extract downloaded torrents.

This is early release, please share your ideas and possible bugs
on github issues or on [hazaah.com](http://www.hazaah.com/programming/utorrent-auto-extractor/).

### Platform and dependencies

1. Tested on Windows 7 and Vista
2. Tested on uTorrent2+
